## **Installation**
```bash
npm i
```
```bash
tsc
```
## **Dev**
**Starts the application in development mode with VITE_ENV=dev.**
```bash
npm run dev
```
## **Production**
**Builds the project for production using Vite.**
```bash
npm run build
```

**Starts the application in production mode.**
```bash
npm run start
```
